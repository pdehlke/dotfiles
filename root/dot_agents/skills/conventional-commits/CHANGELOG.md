# Changelog

## [Unreleased]

### Added
- Requirement that all commit messages be written in English, regardless of the conversation, repository, or code-comment language
- Conventional Commits v1.0.0 skill for Claude Code, enforcing structured types, scopes, breaking-change notation, and message formatting
- Guard against `Co-Authored-By`, copyright, and AI-attribution footers in commit messages

### Fixed
- Corrected the hook configuration structure documented in the README
