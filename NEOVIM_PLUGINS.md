# Neovim Plugins

This document describes each Neovim plugin configured in this setup, organized by functionality.

## Code Editing & Completion

- **[Avante](https://github.com/yetone/avante.nvim)** - AI-powered code assistant that integrates with Claude, allowing you to ask questions and get code suggestions directly within Neovim.

- **[Blink.cmp](https://github.com/saghen/blink.cmp)** - A modern completion engine for Neovim with support for multiple sources including Copilot, featuring a sleek completion menu and documentation window.

- **[Comment](https://github.com/folke/ts-comments.nvim)** - A modern commenting plugin using Treesitter for smart, language-aware code commenting with `gc` and `gcc` keybindings.

- **[Copilot](https://github.com/zbirenbaum/copilot.lua)** - GitHub Copilot integration for Neovim enabling AI-powered code completion and suggestions.

- **[LSP Config](https://github.com/neovim/nvim-lspconfig)** - Configures Language Server Protocol clients for code navigation (go to definition, references, etc.) and code actions.

## Navigation & UI

- **[Barbecue](https://github.com/utilyre/barbecue.nvim)** - A fancy breadcrumb navigation plugin that displays the current code structure (via `nvim-navic`) at the top of your buffer.

- **[Barbar](https://github.com/romgrk/barbar.nvim)** - A customizable buffer line/tabs plugin that displays open buffers at the top of the editor with buffer navigation and management features.

- **[Edgy](https://github.com/folke/edgy.nvim)** - Manages peripheral windows (like sidebars and floating panels) with smart positioning and layout management.

- **[Goto Preview](https://github.com/rmagatti/goto-preview)** - Provides floating window previews for definitions, references, implementations, and type definitions without jumping to them.

- **[Illuminate](https://github.com/RRethy/vim-illuminate)** - Highlights all occurrences of the word under the cursor with smart filtering for certain filetypes.

- **[Neo-tree](https://github.com/nvim-neo-tree/neo-tree.nvim)** - A file explorer sidebar plugin with git integration, file operations, and preview capabilities.

- **[Noice](https://github.com/folke/noice.nvim)** - Enhances Neovim's UI with better looking command line, search, and LSP hover/signature windows.

- **[Telescope](https://github.com/nvim-telescope/telescope.nvim)** - A powerful fuzzy finder for searching files, buffers, grep results, and more with highly customizable layout.

- **[Tiny Code Action](https://github.com/rachartier/tiny-code-action.nvim)** - Displays LSP code actions in a minimal, user-friendly interface with diff previews.

- **[Which-key](https://github.com/folke/which-key.nvim)** - Displays available keybindings in a popup menu to help discover and remember keyboard shortcuts.

## Development Tools

- **[Code Runner](https://github.com/CRAG666/code_runner.nvim)** - Executes code in various programming languages (Python, JavaScript, Rust, Go, etc.) with support for compiler flags and runtime arguments.

- **[Mason](https://github.com/williamboman/mason.nvim)** - A package manager for Neovim that installs and manages LSP servers, linters, formatters, and other tools.

- **[nvim-lint](https://github.com/mfussenegger/nvim-lint)** - A linting framework that integrates with language-specific linters for code quality checks.

- **[Sidekick](https://github.com/folke/sidekick.nvim)** - Integrates with tmux and provides a CLI interface for background tasks and external tool integration.

- **[Toggleterm](https://github.com/akinsho/toggleterm.nvim)** - A terminal integration plugin that provides toggleable floating or split terminals within Neovim.

## Aesthetics

- **[Colorscheme](https://github.com/maxmx03/solarized.nvim)** - The Solarized color scheme with extensive customization options for syntax highlighting and transparency settings.

- **[Lualine](https://github.com/nvim-lualine/lualine.nvim)** - A customizable status line plugin that displays mode, file info, diagnostics, and LSP status.

- **[Mini Animate](https://github.com/echasnovski/mini.nvim)** - Provides smooth animations for window opening/closing and cursor movements.

- **[Status Column](https://github.com/luukvbaal/statuscol.nvim)** - Customizes the status column (line numbers, signs) with support for git signs and DAP breakpoints.

- **[Tiny Inline Diagnostic](https://github.com/rachartier/tiny-inline-diagnostic.nvim)** - Shows diagnostic messages inline in the buffer with smart formatting and wrapping options.

## Utilities

- **[CodeSnap](https://github.com/mistricky/codesnap.nvim)** - Creates beautiful PNG snapshots of your code selections with customizable themes and styling options.

- **[Guess Indent](https://github.com/nmac427/guess-indent.nvim)** - Automatically detects and applies the correct indentation style for your files.

- **[Markdown Preview](https://github.com/iamcco/markdown-preview.nvim)** - Preview Markdown files in a web browser directly from Neovim.

- **[Opencode](https://github.com/sudo-tee/opencode.nvim)** - An AI-powered coding assistant for pair programming and code generation (currently disabled).

- **[SearchBox](https://github.com/VonHeikemen/searchbox.nvim)** - Provides an enhanced search and replace UI with a custom floating window.

- **[Yanky](https://github.com/gbprod/yanky.nvim)** - Enhanced clipboard manager that provides clipboard history and smart paste filtering.
